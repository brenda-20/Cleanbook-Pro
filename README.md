## Project Description

CleanBook Pro is a comprehensive web-based cleaning service management system that streamlines the booking, assignment, and tracking of cleaning services. The system supports three user roles: Customers, Administrators, and Staff.

## Features

- **Customer Portal:**
  - User registration and authentication
  - Browse cleaning services (Automotive, Residential, Furniture)
  - Online booking with date/time selection
  - View booking history and status
  - Payment information

- **Admin Panel:**
  - Dashboard with system statistics
  - Booking management and confirmation
  - Staff assignment to bookings
  - Payment tracking

- **Staff Portal:**
  - View assigned jobs
  - Update job status (In Progress, Completed)
  - Add completion notes
  - Customer contact information

## Technology Stack

- **Frontend:** HTML5, CSS3, Bootstrap 5, JavaScript
- **Backend:** PHP 8.0
- **Database:** MySQL/MariaDB
- **Server:** Apache (XAMPP)

## Installation

1. Clone the repository
2. Import `database.sql` into MySQL
3. Configure database credentials in `config/database.php`
4. Access via `http://localhost/CB/`
